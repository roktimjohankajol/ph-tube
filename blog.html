1. Discuss the scope of var, let, and const !


Var : var is a old variable declear system. we can declear var variable globally. also re-declear within scope

Let : let variable we get from ECMAScript (ES6) they are work alternative to var. But let is a block scope. We can't use globally.

Const : const variable we get from ECMAScript (ES6) they are they are declear the variable. But const is a block scope and not changeable. Also, We can't use globally.

Null : When we need a empty value then we use a null. The null is a primitive type

Undefined : When we declear the variable but not assigned the value. This time we get a Undefined message



3. What do you mean by REST API? !


Rest API : API means Application Programming Interface. One of the key advantages of REST APIs is that they provide flexibility. REST API can handle multiple types of calls, return different data formats and even change structurally with the correct implementation of hypermedia.
